# College Discovery Platform

A full-stack web application to search, compare, and save colleges with real data and user authentication.

## Features
- College listing with search and filters
- College detail page with courses and placements
- Compare 2–3 colleges side-by-side
- User authentication (login/signup)
- Save and view favorite colleges

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: TanStack Server Functions
- Database: PostgreSQL (Lovable Cloud)
- Auth: Built-in authentication

## How it works
- All data is fetched from backend APIs
- No hardcoded frontend data
- Supports filtering, comparison, and user-specific saved items
